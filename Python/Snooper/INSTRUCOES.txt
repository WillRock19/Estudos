1 - Abrir o Arquivo Settings.conf com o bloco de notas.
2 - Alterar o Ip para 127.0.0.1, caso aconteça algum erro, colocar o IP da maquina local(ex: server.socket_host = "xxx.xxx.xxx.xxx").
3 - Alterar o tools.staticdir.root para a pasta que o projeto se localiza.
4 - Executar o Run.Bat

Para testar, coloca localhost:90 no brower, o padrão é a porta 90.
